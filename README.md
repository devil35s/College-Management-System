# College Management System

A modern College Management System built using C# Windows Forms and SQL Server.

## Features
- Student Management
- Attendance Management
- Fees Management
- Faculty Management
- Facility Management
- Sports Management
- Library Management
- Computer Lab Management
- Transport Management
- Reports & Receipt Printing
- Dashboard Analytics

## Technologies Used
- C#
- WinForms
- SQL Server
- ADO.NET
- ClosedXML
- Inno Setup

## Installation Guide

### Prerequisites
Before running the project, make sure the following software is installed:

- Visual Studio 2026
- SQL Server
- .NET Framework 4.8
- Git (optional)

---

### Steps to Run the Project

1. Clone the repository

```bash
git clone https://github.com/devil35s/College-Management-System.git
```

2. Open the solution file in Visual Studio

```text
CollegeManagementSystem.sln
```

3. Restore NuGet packages

4. Open SQL Server Management Studio (SSMS)

5. Run the database script

```text
setup.sql
```

6. Configure database connection string if required

7. Build and run the project

---

## Installer Setup

You can also install the application using the provided setup installer.

### Steps:
1. Download SetupCMS.exe from Releases
2. Install required components if prompted
3. Complete the installation
4. Launch College Management System
5. Run setup.sql
6. Configure DB connection
7. Start application
